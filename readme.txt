== Description ==

SocialVault for Wordpress allows SocialVault users to display their widget on Wordpress websites.

* This plugin is using 3rd party service, circumstances below :

Our plugin may have to use our main website (https://socialvault.com/).
No personal information is used in these calls, only your widget ID will be send threw our calls to https://socialvault.com/.

More information about our privacy policy https://socialvault.com/privacy_policy.

== Installation ==

1. Download form wordpress built-in plugins page.
2. Activate the plugin via the wordpress plugins dashboard page.
3. Insert your widget id (available on socialvault.com/widget/) in "Settings", "SocialVault" and check the "Enable widget" box.
